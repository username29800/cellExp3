documentation for alpha cell (cell_a)
----------
this cell is the alpha cell, which contains the core structure of every cell used in the experiment.
cell_a provides the cell development framework:
1. directory structure
2. core utils for vitality - syn(synthesis; recovery and setup), init
2a. core security utils - mhc(publish new elements)
3. main.py template - write "Hello, world!" to stdout and halts
4. apoptosis script
